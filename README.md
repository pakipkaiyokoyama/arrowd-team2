# Arrowd - VRアクションゲーム

道を矢で作って、積荷をゴールまで運ぶVRアクションゲームです。

## 概要

- **ジャンル**: VRアクション・パズル
- **プラットフォーム**: PC（Unity / VR対応）
- **チーム**: チーム制作（リーダーとして参加）

## ゲーム内容

プレイヤーはVR空間で矢を配置して道を作り、積荷をスタートからゴールまで導きます。空間認識力と戦略的思考が求められるVRならではのゲーム体験です。

## 技術スタック

| 技術 | 用途 |
|------|------|
| **Unity (C#)** | ゲームロジック、VRインタラクション |
| **XR Interaction Toolkit** | VRコントローラー入力・操作 |
| **Unity Input System** | 入力管理 |

## 開発環境

- Unity 2022.3 LTS
- GitHub（チーム開発・バージョン管理）
